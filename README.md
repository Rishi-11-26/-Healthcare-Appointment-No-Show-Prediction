# -Healthcare-Appointment-No-Show-Prediction
 Healthcare Appointment No-Show Prediction
 This project predicts whether a patient will miss their healthcare appointment and provides insights to reduce no-show rates.

Files Included

appointments_sample.csv – Cleaned dataset used for modelling and Power BI

train_no_show_model.py – Python script to train the Decision Tree model

no_show_model.pkl – Trained prediction model

optimization_recommendations.txt – Suggestions to reduce no-shows
